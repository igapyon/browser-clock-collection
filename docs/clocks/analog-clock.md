# Analog Clock

## 概要
クラシックなアナログ時計をブラウザで表示します。秒針・分針・時針が現在時刻に同期して動きます。

## 表示内容
- 12時間表示の文字盤
- 分目盛り（5分ごとに強調）
- 時針／分針／秒針

## 振る舞い
- 1秒ごとに時刻を更新
- 秒の境界に合わせて更新（ズレが蓄積しにくい）

## 動作環境
- モダンブラウザ（Chrome / Edge / Safari / Firefox）
- 外部ライブラリ: SVG.js（CDN）

## 使い方
- `src/clocks/analog/analog-clock.html` を直接ブラウザで開く

## 変更の目安
- サイズ: ページ内の `#mount` 幅を調整
- 色味: テーマ色の変更
