# Flip Date

## 概要
フリップ式の日付表示です。年・月・日・曜日を大きく読みやすく表示します。

## 表示内容
- 年（YYYY）
- 月／日（MM / DD）
- 曜日（日本語）

## 振る舞い
- 1秒ごとに日付を監視
- 日付が変わったタイミングでフリップ更新
- 月や年が変わる場合も適切に更新

## 動作環境
- モダンブラウザ（Chrome / Edge / Safari / Firefox）
- 外部ライブラリ: SVG.js（CDN）

## 使い方
- `src/clocks/flip-date/flip-date-clock.html` を直接ブラウザで開く

## 変更の目安
- サイズ: ページ内の `#mount` 幅を調整
- 曜日位置: 右側のラベル位置を調整
